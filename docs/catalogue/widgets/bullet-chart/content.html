A bullet chart as designed by Stephen Few showing a single value against a target with a quantitative scale and three qualitative states that may resemble ranges defining poor, satisfactory or good. Bullet charts are useful to visualize the state some metric in comparison to goals and help interpret the value easily. The forecast value can be useful when estimating whether the target will be reached given the current performance.
