Placeholder

api reference

Placeholder()
Component implementing the placeholder feature. The placeholder is a blank div with a message in the middle that replaces the widget. It is useful when no data is available, or when the widgets needs to be hidden/shown dynamically. When this component is available for a widget, its API is exposed directly via the widget's own namespace.
Placeholder.placeholder([content[, duration]])
Shows/hides the placeholder. If no placeholder content is provided, the widget is recovered.

Parameters

NameTypeDescription
contentstringoptionalContent of the placeholder. Can be HTML formatted. If omitted, the placeholder is removed. Note that the content can be an empty string in which case the widget is simply hidden.
durationnumberoptionalDuration of the placeholder animation in ms. Default value is 400.

Returns

Widget
Reference to the Widget's API.
