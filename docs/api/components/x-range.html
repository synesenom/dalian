XRange

api reference

XRange()
Component implementing the Y range constraints. This component enables setting the horizontal min/max boundaries for a chart. When this component is available for a widget, its API is exposed via the .xRange namespace.
XRange.max(value[, stretch])
Sets the upper boundary for the X axis.

Parameters

NameTypeDescription
valuenumber, nullThe upper boundary to set. If null, the plot's own upper boundary is used.
stretchnumberoptionalScaling factor to stretch the upper boundary with. If not zero, the upper boundary is expanded with the specified fraction of the entire X domain of the real plot area. Default value is 0.

Returns

Widget
Reference to the Widget's API.
