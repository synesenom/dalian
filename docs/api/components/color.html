Color

api reference

Color()
Component implementing various color scheming features. When this component is available for a widget, its API is exposed directly via the widget's own namespace.
Color.color([policy[, size]])
Sets the color policy. Supported policies:
  • Default color policy (no arguments): the default color scheme is used which is a modification of the qualitative color scheme Set 1 from Color Brewer.
  • Single color or shades of a color (passing string): Either the specified color is used for all plots or a palette is generated from its shades (see size).
  • Custom color mapping (passing an Object): each plot has the color specified as the value for the property with the same name as the plot's key.

Parameters

NameTypeDescription
policystring, ObjectoptionalColor policy to set. If not specified, the default policy is set.
sizenumberoptionalNumber of colors that need to be generated if policy is set to a single color. If not set, the color specified for policy is used for all plots.
