Pin

api reference

Pin()
Component implementing the pin feature. A pin is a vertical line with a circle on the top and an optional text label as description. When this component is available in a widget, it is accessible through the .pin namespace.
Pin.add(id, position[, options[, duration]])
Adds a pin to the chart. If the marker ID already exists, no further markers are added.

Parameters

NameTypeDescription
idstringUnique identifier of the pin.
positionnumberValue to add pin to.
optionsObjectoptionalPin design options. Supported values:
  • string color: Pin color.
  • number size: Size of the pin head in pixels.
  • number height: Pin height relative to the Y range.
  • string text: Label to add to the pin. The label is visible when hovering over the pin.
 Default value is {}.
durationnumberoptionalDuration of the animation of adding the pin. Default value is 400.

Returns

Object
The object representing the pin. This object contains the DOM group containing the pin and update/remove methods.
Pin.remove([id[, duration]])
Removes one or all pins from the chart.

Parameters

NameTypeDescription
idstringoptionalIdentifier of the pin to remove. If pin with the specified identifier does not exist, no change is applied. If it is not specified, all pins are removed from the current chart. Default value is undefined.
durationnumberoptionalDuration of the remove animation. Default value is 400.
