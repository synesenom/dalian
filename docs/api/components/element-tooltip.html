ElementTooltip.PointTooltip()
Component implementing the element tooltip feature. The element tooltip is a pre-formatted tooltip that takes a data point and displays it in the tooltip with a colored left border. Can be used in widgets where the tooltip displays info for a specific element in the widget (hence the name), such as bar widgets or pie widgets. It inherits all methods from the Tooltip component. The tooltip consists of the name of the current element (which can be formatted using titleFormat ) and the available coordinates of the element (X, Y or both). The coordinate values can be formatted using the corresponding xFormat and yFormat methods.
