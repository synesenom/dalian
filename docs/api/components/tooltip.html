Tooltip

api reference

Tooltip()
Component implementing the tooltip feature. When this component is available for a widget, its API is exposed via the .tooltip namespace.
Tooltip.ignore(keys)
Sets the array of keys that are ignored by the tooltip. Ignored keys are not shown in the tooltip and they don't have plot markers.

Parameters

NameTypeDescription
keysstring[]Keys of plots to ignore in the tooltip.

Returns

Reference to the Widget API.

Tooltip.rowFormat([format])
Sets the format for the tooltip rows when the content does not specify the coordinate. An example is the content of the ElementTooltip.

Parameters

NameTypeDescription
formatFunctionoptionalFunction to use as the formatter. May take one parameter which is the row value of the tooltip. Default value is x => x.
