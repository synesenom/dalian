LineChart.addMarker(id, key, start, end, label[, duration])
Adds a trend marker to the chart. A trend marker is a labeled pair of dots indicating changes in the plot. If the marker ID already exists, no further markers are added.

Parameters

NameTypeDescription
idstringUnique identifier of the trend marker.
keystringKey of the plot to add the trend marker to.
startnumberStarting (left side) value of the trend marker.
endnumberEnding (right side) value of the trend marker.
labelstringLabel to display on the marker.
durationnumberoptionalDuration of the animation of adding the marker. Default value is 700.

Returns

LineChart.colors([policy[, size]])
Sets the color policy for the plots. Supported policies:
  • Default color policy (no arguments): the default color scheme is used which is a modification of the qualitative color scheme Set 1 from Color Brewer.
  • Single color or shades of a color (passing string): Either the specified color is used for all plots or a palette is generated from its shades (see size).
  • Custom color mapping (passing an Object): each plot has the color specified as the value for the property with the same name as the plot's key.

Parameters

NameTypeDescription
policystring, ObjectoptionalColor policy to set. If not specified, the default policy is set.
sizenumberoptionalNumber of colors that need to be generated if policy is set to a single color. If not set, the color specified for policy is used for all plots.

Returns

LineChart.data(plots)
Set/updates the data that is shown in the line chart.

Parameters

NameTypeDescription
plotsObject[]Array of objects representing the lines to show. Each plot has two properties:
  • string name: Name of the plot.
  • Object[] values: Plot data.
The values property is an array of objects of the following structure:
x number
X coordinate of the data point.
y number
Y coordinate of the data point.
lo number
Lower error of the data point. optional
hi number
Upper error of the data point. optional

Returns

Enables/disables description for the line chart. A description is a small tooltip that is bound to the context menu (also disables default event handler). The description disappears once the mouse leaves the chart. If called without argument, description is disabled.

LineChart.lineStyle([policy])
Sets the line style policy. Supported policies:
  • Default line style policy (no arguments): the default line style is used which is solid for all plots.
  • Single line style (passing string): The specified line style is used for all plots. Supported styles are: solid, dashed, dotted.
  • Custom line style mapping (passing an Object): each plot has the line style specified as the value for the property with the same name as the plot's key.

Parameters

NameTypeDescription
policystring, ObjectoptionalLine style policy to set. If not specified, the default policy is set.

Returns
