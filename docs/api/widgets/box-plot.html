BoxPlot(name[, parent])
The box plot widget. As a chart, it extends the Chart component, with all of its available APIs. Furthermore, it extends the following components:
  • BottomAxis
  • ElementTooltip The default entries shown in the tooltip are: median, q1, q3, lower whisker (as low), upper whisker (as high), number of mild outliers, number of extreme outliers.
  • Highlight Boxes can be highlighted by passing their names as specified in the data array.
  • LeftAxis
  • Opacity

Parameters

NameTypeDescription
namestringName of the chart. Should be a unique identifier.
parentstringoptionalQuery selector of the parent element to append widget to. Default value is body.

BoxPlot.data(plots)
Set/updates the data that is shown in the box plot. Each bar is a plot group in itself, so all methods that operate on plot groups are applied on the box level.

Parameters

NameTypeDescription
plotsObject[]Array of objects representing the boxes to show. Each box has two properties:
  • string name: Category name.
  • number value: Category value.
The values property is an object with the following structure:
median number
The sample median.
q1 number
Lower quantile of the sample.
q3 number
Upper quantile of the sample.
whiskers Object
Object containing two numbers: the lower and upper whiskers. Meaning of these can vary.
outliers Object
Object containing two arrays of numbers: the list of mild and extreme outliers
