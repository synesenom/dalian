AreaChart

api reference

AreaChart(name[, parent])
The area chart widget. Being a chart, it extends the Chart component, with all of its available API. FUrthermore it extends the following components: BottomAxis , Highlight , LeftAxis , Opacity , PointTooltip .

Parameters

NameTypeDescription
namestringName of the chart. Should be a unique identifier.
parentstringoptionalParent element to append widget to. Default value is body.
AreaChart.data(plots)
Set/updates the data that is shown in the area chart.

Parameters

NameTypeDescription
plotsObject[]Array of objects representing the area plots to show. Each plot has two properties:
  • string name: Name of the plot.
  • Object[] values: Plot data.
The values property is an array of objects of the following structure:
x number
X coordinate of the data point.
y number
Y coordinate of the data point.

Returns

AreaChart
The AreaChart itself.
AreaChart.highlight([keys[, duration]])
Highlights a single plot or multiple plots.

Parameters

NameTypeDescription
keysstring, string[], nulloptionalSingle key or array of keys identifying the plots to highlight. If key is null or undefined, the highlight is removed (all plots become visible).
durationnumberoptionalDuration of the highlight animation in ms. Default value is 700.

Returns

AreaChart
The AreaChart itself.
AreaChart.smoothing([on])
Enables/disables polygon smoothing.

Parameters

NameTypeDescription
onbooleanoptionalWhether to enable polygon smoothing. If not specified, smoothing is disabled. Default value is false.
