ScatterPlot

api reference

ScatterPlot(name[, parent])
The scatter plot widget. Being a chart, it extends the Chart component, with all of its available API. Furthermore, it extends the following components: BottomAxis , ElementTooltip , Highlight , LeftAxis , Opacity , XRange , YRange .

Parameters

NameTypeDescription
namestringName of the chart. Should be a unique identifier.
parentstringoptionalParent element to append widget to. Default value is body.
ScatterPlot.size([value])
