BarChart

api reference

BarChart(name[, parent])
The bar chart widget. Being a chart, it extends the Chart component, with all of its available API. Furthermore, it extends the following components: BottomAxis , LeftAxis , ElementTooltip .

Parameters

NameTypeDescription
namestringName of the chart. Should be a unique identifier.
parentstringoptionalParent element to append widget to. Default value is body.
BarChart.data(plots)
Set/updates the data that is shown in the bar chart.

Parameters

NameTypeDescription
plotsObject[]Array of objects representing the bars to show. Each bar has two properties:
  • string name: Category name.
  • number value: Category value.

Returns

BarChart
The BarChart itself.
BarChart.highlight([keys[, duration]])
Highlights a single bar or multiple bars.

Parameters

NameTypeDescription
keysstring, string[], nulloptionalSingle key or array of keys identifying the bars to highlight. If key is null or undefined, the highlight is removed (all bars become visible).
durationnumberoptionalDuration of the highlight animation in ms. Default value is 700.

Returns

BarChart
The BarChart itself.
BarChart.horizontal(on)
Converts the bar chart to a horizontal bar chart. Note that this method does not swap the axis labels.

Parameters

NameTypeDescription
onbooleanWhether the bar chart should be horizontal.
