Slider(name[, parent])
The slider control widget. Inherits all API from the Widget component. The slider is constrained to a rectangle defined by the width/height and the margins. It is vertically centered. The scale ticks are implemented using BottomAxis , making all the API for that component available for the slider.

Parameters

NameTypeDescription
namestringName of the widget. Should be a unique identifier.
parentstringoptionalSee Widget for description. Default value is body.
Slider.callback([callback])
Binds a callback to the slider. Note that this is called every time the handle is dragged (you may want to debounce it).

Parameters

NameTypeDescription
callbackFunctionoptionalCallback to bind when the handle is dragged. The current value is passed as parameter. Default value is null.

Returns

Slider
Reference to the Slider API.

Examples

// Print current value to console.
const slider = dalian.Slider('my-control')
  .callback(console.log)
  .render()

// Remove callback.
slider.callback()
  .render()
